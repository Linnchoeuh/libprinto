# printo
>A printf like lib. 
##Features
This lib has to major function : printo and 
This function is able to print: 
  -int/long (%d)
  -float/double (%f)
  -pointer (%p)
  -char (%c)
  -str (%s)
  -boolean (%B)
The usage is very similar to printf, but you also have to give at first parameter the output (where you want to write it)
